# LLM-Based AI Projects

## Overview
This repository contains a collection of **LLM-powered AI applications** built using
Python, Streamlit, and OpenAI APIs.  
Each project focuses on a different real-world use case of Large Language Models,
including content generation, education, analysis, and personal assistance.

---

## Projects Included

### 1. Blog Post Generator
AI-powered tool that generates structured, SEO-friendly blog posts from a given topic.

**Tech:** Python, Streamlit, OpenAI  
Folder: `blog-post-generator`

---

### 2. Text Summarizer
Summarizes long text into concise and meaningful content.

**Tech:** Python, Streamlit, OpenAI  
Folder: `text-summarizer`

---

### 3. AI Tutor (Math, CS, Physics)
An intelligent tutor that explains (Maths, Physics, Computer Science) academic concepts step-by-step.

**Tech:** Python, Streamlit, OpenAI  
Folder: `ai-tutor`

---

### 4. Sentiment Analysis Tool
Classifies text sentiment as Positive, Negative, or Neutral.

**Tech:** Python, Streamlit, OpenAI  
Folder: `sentiment-analysis`

---

### 5. Personal AI Assistant
A context-aware assistant that responds only using predefined personal data.

**Tech:** Python, Streamlit, OpenAI  
Folder: `personal-assistant`

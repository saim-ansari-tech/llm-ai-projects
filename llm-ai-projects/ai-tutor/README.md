# AI Tutor (Math, CS, Physics)

## Overview
An AI-powered tutoring application that helps users understand concepts in
Mathematics, Computer Science, and Physics using conversational explanations.

---

## Features
- Subject-specific tutoring
- Step-by-step explanations
- Secure API key handling

---

# Personal AI Assistant (Context-Aware)

## Overview
A context-aware personal AI assistant that answers questions strictly
based on predefined personal data. If the question is outside the
provided knowledge, the assistant responds accordingly.

---

## Features
- Controlled knowledge-based responses
- Prevents hallucinations
- Personal data-driven assistant
- Secure API key handling

---

## Use Case
- Personal portfolio assistant
- Resume-aware chatbot
- Knowledge-restricted AI systems

---

## How to Run
1. Install dependencies:
```bash
pip install -r requirements.txt

2. Add API key in .env
OPENAI_API_KEY=your_api_key_here

3. Run:
Streamlit run app.py
